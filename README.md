# 舒尔特方格
德国心理医生沃尔特·舒尔特在图宾根大学工作期间（1962-1972），发明了舒尔特方格。舒尔特方格最初是为了飞行员开发的，其目标是为了提高飞行员处理突发情况的能力，即，短时间内的反应能力。例如，我国飞行员都会进行这个训练，他们的平均速度是6.25秒，杨利伟是我国用时最短的，时间为3.04秒（5*5方格）。舒尔特方格，除了能训练飞行员，还能训练驾驶员。—— 言论来自网络
